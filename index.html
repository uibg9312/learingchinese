Orden Básico Sujeto → (Tiempo/Lugar) → Verbo → Objeto 我今天去学校。 Wǒ jīntiān qù xuéxiào. Yo hoy voy a la escuela. Orden SVO general.

Detalles: El orden de las palabras en chino es crucial. Los modificadores (tiempo, lugar, manera) generalmente preceden al verbo.

Ejemplo Adicional: 他昨天在北京工作。(Tā zuótiān zài Běijīng gōngzuò.) - Él ayer trabajó en Pekín.
Negación 不 (bù) / 没 (méi) 他没去学校。/ 我不吃肉。 Tā méi qù xuéxiào. / Wǒ bù chī ròu. Él no fue a la escuela. / No como carne. 不: general/futuro; 没: pasado/没有 (no tener).

不 (bù): Niega acciones habituales, presentes, futuras y adjetivos. Cambia a tono 2 (bú) antes de un tono 4.

没 (méi): Niega acciones pasadas completadas y el verbo 有 (yǒu - tener/haber). A menudo se usa como abreviatura de 没有 (méiyǒu).

Ejemplo Adicional (bù): 这个不贵。(Zhège bù guì.) - Esto no es caro.

Ejemplo Adicional (méi): 我昨天没看见他。(Wǒ zuótiān méi kànjiàn tā.) - No lo vi ayer.
Pregunta Sí/No ...吗? / V-不-V? 你喜欢咖啡吗？/ 你去不去学校？ Nǐ xǐhuān kāfēi ma? / Nǐ qù bù qù xuéxiào? ¿Te gusta el café? / ¿Vas o no vas a la escuela? 吗 al final / Repetir verbo con 不 en medio.

吗 (ma): La forma más simple y común. Se añade al final de una oración afirmativa.

V-不-V: Más común en el habla. Solo funciona con verbos monosilábicos o bisilábicos (el primero se repite). No se usa con 吗.

Ejemplo Adicional (V-不-V): 他高不高？(Tā gāo bù gāo?) - ¿Es alto (él)?

Nota: Con adjetivos, la estructura es Adj-不-Adj.
Partícula 的 (de) Posesión / Modificador + 的 + Sustantivo 我的书 / 红色的苹果 Wǒ de shū / Hóngsè de píngguǒ Mi libro / Manzana roja Une atributos al sustantivo.

Usos Principales:

  • - Posesión: 老师的学生 (lǎoshī de xuéshēng) - El estudiante del profesor.
  • - Modificación (Adjetivo + de + N): 聪明的孩子 (cōngmíng de háizi) - Niño inteligente.
  • - Modificación (Frase verbal + de + N): 我买的票 (wǒ mǎi de piào) - El boleto que compré.

Omisión: A veces se omite con relaciones cercanas (我妈妈 - wǒ māma) o atributos inherentes (木头桌子 - mùtou zhuōzi - mesa de madera).
Partícula 了 (le) Acción completada / Cambio de estado 他走了。/ 下雨了。 Tā zǒule. / Xià yǔ le. Se fue. / Empezó a llover (o está lloviendo ahora). Marca pasado o cambio.

Uso 1 (Después del verbo): Indica que una acción se completó.

Ejemplo: 我吃了饭。(Wǒ chīle fàn.) - Comí.

Uso 2 (Al final de la oración): Indica un cambio de estado o una nueva situación.

Ejemplo: 他是老师了。(Tā shì lǎoshī le.) - Ahora es profesor (antes no lo era).

Nota: Puede haber dos 了 en una oración: 我吃了饭了。(Wǒ chīle fàn le.) - Ya comí (acción completada + estado actual).
Partícula 过 (guò) Experiencia pasada 我去过中国。 Wǒ qùguo Zhōngguó. He estado en China (alguna vez). Indica haber hecho algo al menos una vez.

Diferencia con 了 (le):

  • - **了 (le):** Acción completada, a menudo con relevancia en el presente o secuencia. 我看了那本书。(Wǒ kànle nà běn shū.) - Leí ese libro (acción terminada).
  • - **过 (guò):** Experiencia en algún momento del pasado, sin importar cuándo o si se repitió. 我看过那本书。(Wǒ kànguo nà běn shū.) - He leído ese libro (tengo la experiencia de haberlo leído).

Negación: Se niega con 没(有) + Verbo + 过. 他没去过。(Tā méi qùguo.) - Él no ha ido (nunca).
Clasificadores (个, 本, 只...) Número + Clasificador + Sustantivo 一个人 / 三本书 / 一只猫 Yí ge rén / Sān běn shū / Yì zhī māo Una persona / Tres libros / Un gato Obligatorios entre número/demostrativo y sustantivo.

Uso: Siempre se usan después de números (excepto a veces 'uno') y demostrativos (这 zhè - este, 那 nà - ese).

Comunes:

  • - **个 (ge):** El más común, para personas, objetos generales.
  • - **本 (běn):** Para cosas encuadernadas (libros, revistas).
  • - **只 (zhī):** Para animales (gatos, perros, pájaros), partes de un par (un zapato).
  • - **张 (zhāng):** Para objetos planos y delgados (papel, mesas, camas, boletos).
  • - **辆 (liàng):** Para vehículos con ruedas (coches, bicicletas).
  • - **杯 (bēi):** Para tazas/vasos de algo (una taza de té).

Ejemplo con demostrativo: 这本书 (Zhè běn shū) - Este libro.
Estructura 把 (bǎ) Sujeto + 把 + Objeto + Verbo + Complemento/了 我把书放在桌子上了。 Wǒ bǎ shū fàng zài zhuōzi shàng le. Puse el libro sobre la mesa. Enfatiza el efecto sobre el objeto.

Función: Mueve el objeto directo antes del verbo para enfatizar qué se le hizo al objeto (disposición).

Requisitos:

  • - El objeto debe ser definido o específico (no "un libro", sino "el libro" o "ese libro").
  • - El verbo debe indicar una acción que afecta al objeto.
  • - Generalmente requiere un complemento después del verbo (de resultado, dirección, 了, etc.).

Ejemplo Adicional: 请把门打开。(Qǐng bǎ mén dǎkāi.) - Por favor, abre la puerta.

Negación: Se niega antes de 把: 我没把作业做完。(Wǒ méi bǎ zuòyè zuòwán.) - No terminé la tarea.
Complementos (Resultado) Verbo + Complemento Resultativo (完, 到, 见, 好...) 我做完了作业。/ 我看见了她。 Wǒ zuòwánle zuòyè. / Wǒ kànjiànle tā. Terminé la tarea. / La vi. Indican el resultado de la acción verbal.

Función: Se adjuntan directamente al verbo para especificar el resultado.

Comunes:

  • - **完 (wán):** terminar, completar (吃完 chīwán - terminar de comer).
  • - **到 (dào):** alcanzar, lograr (找到 zhǎodào - encontrar; 买到 mǎidào - lograr comprar).
  • - **见 (jiàn):** percibir sensorialmente (看见 kànjiàn - ver; 听见 tīngjiàn - oír).
  • - **好 (hǎo):** completar satisfactoriamente (做好 zuòhǎo - hacer bien/terminar).
  • - **懂 (dǒng):** entender (听懂 tīngdǒng - entender al oír; 看懂 kàndǒng - entender al leer).
  • - **错 (cuò):** hacer incorrectamente (写错 xiěcuò - escribir incorrectamente).

Negación: Se usa 没(有) antes del verbo: 我没听懂。(Wǒ méi tīngdǒng.) - No entendí (al oír).
Complementos (Dirección) Verbo + 来/去 (Simple) / Verbo + 上/下/进/出... + 来/去 (Compuesto) 他上来了。/ 他跑进去了。 Tā shànglai le. / Tā pǎo jìnqu le. Subió (hacia el hablante). / Entró corriendo (alejándose). Indican la dirección del movimiento.

来 (lái):** Movimiento hacia el hablante.

去 (qù):** Movimiento alejándose del hablante.

Simples: 上来 (shànglai - subir hacia aquí), 下去 (xiàqu - bajar hacia allá), 进来 (jìnlai - entrar hacia aquí), 出去 (chūqu - salir hacia allá), 回来 (huílai - volver aquí), 过去 (guòqu - ir allá/pasar), 起来 (qǐlai - levantarse/empezar).

Compuestos: Combinan un verbo de acción con un complemento de dirección simple: 跑上来 (pǎo shànglai - subir corriendo hacia aquí), 拿下去 (ná xiàqu - bajar algo [llevándolo] hacia allá).

Con Objeto: Si el objeto es un lugar, va entre el verbo y 来/去 (他回家去了 Tā huí jiā qù le - Volvió a casa). Si es un objeto portable, puede ir entre el verbo y 来/去 o después (他拿出书来 Tā ná chū shū lai / 他拿出来书 Tā ná chūlai shū - Sacó el libro).
Complementos (Potencial) Verbo + 得/不 + Complemento Resultativo/Direccional 我看得懂。/ 我听不见。 Wǒ kàn de dǒng. / Wǒ tīng bu jiàn. Puedo entender (al leer). / No puedo oír (o no oigo). Indica si es posible lograr el resultado/dirección.

Forma Afirmativa: Verbo + 得 (de) + Complemento (Resultativo o Direccional).

Ejemplo: 你拿得动吗？(Nǐ ná de dòng ma?) - ¿Puedes moverlo/cargarlo?

Forma Negativa: Verbo + 不 (bu) + Complemento (Resultativo o Direccional).

Ejemplo: 这个字我写不好。(Zhège zì wǒ xiě bu hǎo.) - No puedo escribir bien este carácter.

Nota: Es la forma de preguntar/negar la posibilidad de un verbo con complemento. No confundir con la negación de la acción pasada (que usa 没).

Resumen Gráfico Final

【 SUJETO 】→【 (Tiempo/Lugar) 】→【 (Adv./Modal/把+O) 】→【 VERBO 】→【 (Complementos/了/过) 】→【 OBJETO 】+ (Partículas Finales)

+ Pregunta Sí/No: + 吗?
+ Pregunta V/No V: + V + 不 + V?
+ Pregunta Potencial: + V + 得/不 + Complemento?
+ Negación: 不 (bù) / 没 (méi) + Verbo
+ Posesión/Atributo: + 的 (de)
+ Pasado/Cambio: + 了 (le)
+ Experiencia: + 过 (guò)
+ En progreso: 在 / 正在 (zài / zhèngzài) + Verbo + (呢 ne)
+ Comparativo: A + 比 (bǐ) + B + Adj.
+ Pasiva: Receptor + 被 (bèi) + (Agente) + Verbo
+ Disposición: Sujeto + 把 (bǎ) + Objeto + V + Comp.
+ Clasificadores: Num/Dem + Clasificador + Sust.
+ Complementos: V + (得/不) + Comp. (Resultado/Dirección)
